Just a basic typewriter animation that you can drop into your web page. The specific actions you want to take are communicated via chainable api.

 
			var obj = new Typewriter('root', {"fontFamily": "'Arimo', sans-serif", loop: true});
			
			
			obj
			  .typeCharacters('Hella')
			  .deleteCharacters(1)
			  .typeCharacters('o, my name i Andy.')
			  .moveLeft(6)
			  .typeCharacters('s')
			  .moveRight(6)
			  .pauseFor(2000)
			  .deleteCharacters('delete all')
			  .start();
Method Use Case
start() Place at the end of chain to trigger animation.
typeCharacters(string) Place string as argument in this method for typewriter content
deleteCharacters(number|string) Place number or string('delete all') to delete that number of characters. If you use number over that which is available, it will simply remove the max number of characters
moveLeft(number) move cursor n number of spaces to the left
moveRight(number) move cursor n number of spaces to the right
pauseFor(ms) Pause next animation till certain time in milliseconds
